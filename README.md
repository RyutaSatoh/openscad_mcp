# OpenSCAD MCP Server

An MCP server that allows agents to render OpenSCAD files to images (multi-view) and export them to STL.

## Features

- **render_views**: Generates a composite image with Top, Front, Right, and Isometric views.
- **export_stl**: Compiles the SCAD file into an STL mesh.

## Requirements

- OpenSCAD installed and available in PATH.
